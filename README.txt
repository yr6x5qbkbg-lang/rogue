1. ゲームの目的
	地下に潜りながら落ちているゴールドを数多く集めましょう。15Fにいるドラゴンを倒すとゲームクリアです。
	
2.	必要環境
	Python3.0以降
	tcodライブラリのインストール
	
3. 操作説明
	十字キー(←↓↑→) : 各方向移動
	h, j, k, l : 左、下、上、右移動
	y, u, b, n : 斜め移動。
	g : 床に落ちているアイテムを拾う。
	i : インベントリ。ここからアイテムを選択し、使用・装備する。
	d : 持ち物を捨てる。
	v : メッセージのログを表示。
	c : プレイヤーのステータスを表示。
	> : 階段を降りる。
	Enterキー : スクロールなどで対象の選択が必要な場合の決定。
	Escキー : ゲームを終了する。ゲームオーバーでなければメイン画面よりコンティニュー可能。
	r : メイン画面、ゲームオーバー時にランキング画面を表示。
	共通 : 何かを選択する場合はキーボードのアルファベットが対応している。
	
4. 画面説明
	A~Z : エネミー、敵。
	＊ : ゴールド。最大50g。
	! : ポーション。1種類のみ。
	/ : 武器。+1~+3までの強化あり。
	~ : スクロール、魔法の巻物。
	[ : 防具。強化はなし。

5. ルール説明
	・こちらが行動すると相手ターンとなり相手も行動します。
	・相手に向かってぶつかると攻撃。
	
6. 細かい仕様説明
	・ゲームオーバー時にrキーを押してください。ランキングが表示されます。(メッセージ改修予定)
	・移動の他に拾う(g)、使う、装備する/はずす、捨てる(d)、は1ターンの行動とみなされます。
	・空振りや壁に衝突でターンの消費はありません。
	・confusion scrollは対象を選択してEnterで発動。
	・fireball scrollは範囲を移動選択してEnterで発動。
	・ドラゴンに対してのみscrollのダメージが半減。
	・ゲームクリアした場合、持ち物が換金されボーナスとしてスコアに加算される。
	・クリアボーナスはポーションが20g、防具が30g、他はすべて10g。ただし初期装備は0g。
	・武器に関してはsword+3が次のクラスのsword(sxxxx_sword)より弱いというわけではないため実際に装備して、ステータスを確認するのが無難。上昇値表記を改善予定。
	・他、ネタバレに関する設定等はSPOILER.txtに記載。攻略に必須ではありません。
